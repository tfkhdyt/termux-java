## Requirements
- <a href="https://play.google.com/store/apps/details?id=com.termux">Termux</a>
- `git (pkg install git)`

## Instalasi
```Bash
git clone https://github.com/tfkhdyt/termux-java.git && cd termux-java && ./install.sh
```
## Cara penggunaan
```Bash
jv [nama_file.java]
# contoh
jv Pinjaman.java
```
